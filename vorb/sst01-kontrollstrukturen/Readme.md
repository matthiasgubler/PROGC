_Matthias Gubler (gublema6) & Daniele Colombo (colomdan)_

Nach dem Kompilieren kann das Modul via ./kontrollstrukturen aufgerufen werden können.
Es können zwei Parameter mitgeben werden:
Parameter 1 = Zahl zwischen 0 und 255
Parameter 2 = Zu manipulierendes Bit (start ist das 0'te bit)

z.B:
./kontrollstrukturen 135 2
./kontrollstrukturen 17

Je nachdem ob 1 oder 2 Parameter eingegebn werden, werden unterschiedliche Operationen ausgeführt.
